# GNU-EFI Hello World

A simple hello world using GNU-EFI.

## Startup

If you want to use this project as a template, you will need to update Makefile according to your environment and project.

## Make targets

```shell
$ make # Just compile.
$ make clean # Clean.
$ make qemu # Run qemu to test your EFI.
```
